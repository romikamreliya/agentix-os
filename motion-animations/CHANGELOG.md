# Changelog

All notable changes to this skill are documented here. Newest first.

## [2.0.0] - 2026-07-02 — major
### Changed (breaking)
- **Renamed the skill `react-motion-animations` → `motion-animations`** and
  broadened its scope beyond React. The directory, `name`, and trigger scope
  changed, so prior React-only usage no longer describes the whole skill.
- `SKILL.md` — new "Integration methods" table (React / plain JS / CDN / other
  frameworks); description, title, intro, setup, and workflow (esp. new step 2,
  "Determine the integration method") are now framework-agnostic. Added the
  required "When & why to use" section.
### Added
- `references/vanilla-and-cdn.md` — the framework-agnostic JS API (`animate`,
  `scroll`, `inView`, `hover`, `press`, `stagger`, `spring`), a copy-paste CDN
  single-HTML page, vanilla equivalents of the common patterns, and notes for
  Vue (`motion-v`), Svelte, Angular, and Astro.
- `references/performance-and-a11y.md` — vanilla `matchMedia` reduced-motion
  guard, `motion/mini` bundle note, and vanilla SSR/flash guidance.
- Cross-links from `api-cheatsheet.md`, `patterns.md`, and `use-cases.md` to the
  vanilla reference; use-cases noted as method-agnostic.

Source: user request — Motion is not React-only; cover React, CDN single-HTML,
and other frameworks/environments the library supports.

## [1.1.0] - 2026-07-02 — minor
### Added
- `references/use-cases.md` — guidance on **when and where** to use motion and
  each of the 12 patterns: a pattern → use case → applicable UI components table,
  a trigger-based quick lookup, and intensity guidance by surface (marketing vs.
  app vs. data/admin), plus when to skip/minimize animation.
### Changed
- `SKILL.md` — reference list now includes `use-cases.md`; workflow step 1 now
  confirms motion fits and matches surface intensity, and step 3 selects the
  pattern (via use-cases) before the mechanism.

Source: user request to specify where/when the skill should be used, with
recommended use cases and applicable UI components per animation pattern.

## [1.0.0] - 2026-07-02 — initial
### Added
- Initial release generated by ai-skill-creator.
- `SKILL.md` with a 7-step workflow for adding Motion (`motion/react`) animations
  to React apps.
- `references/api-cheatsheet.md` — core API: `motion` component, props,
  transitions (spring/tween), variants + `stagger`, gestures, scroll, layout,
  `AnimatePresence`, `useAnimate`, MotionValues, reduced motion.
- `references/patterns.md` — 12 copy-paste recipes (fade-in, hover/tap button,
  staggered list, scroll reveal, scroll-linked progress, modal, page transition,
  layout & shared-element, drag, accordion, presets).
- `references/performance-and-a11y.md` — cheap-property guidance, timing, bundle
  size (`LazyMotion`/`m`), and `prefers-reduced-motion` handling.

Source inputs: Motion for React documentation — https://motion.dev/docs
(react-quick-start, react-animation, react-transitions, react-gestures).
