# Changelog

All notable changes to this skill are documented here. Newest first.

## [1.0.0] - 2026-07-02 — initial
### Added
- Initial release generated by ai-skill-creator.
- `SKILL.md` — a 7-step triage workflow (read → extract signals → classify →
  summarize → assess clarity → clarify-if-needed → emit tag block), the required
  "When & why to use" section, and core rules (classify before acting, one primary
  category, clarify only when it changes the outcome).
- `references/categories.md` — the taxonomy (task / research / code-change / chat
  / documentation / other) with signals and disambiguation, context tags, and the
  compact classification output format.
- `references/clarifying-questions.md` — when to ask, and how to ask via the
  `AskUserQuestion` tool with suggested options plus an automatic Other choice.
- `LICENSE.txt` — MIT.

Source: user request — analyze/understand the prompt, categorize it, summarize
it, tag it before processing, and ask option-based clarifying questions (with an
Other option) when the prompt is unclear.
