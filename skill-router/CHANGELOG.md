# Changelog

All notable changes to this skill are documented here. Newest first.

## [1.0.0] - 2026-07-02 — initial
### Added
- Initial release generated by ai-skill-creator.
- `SKILL.md` — a 7-step router workflow (inventory → understand → match & rank →
  decide → confirm-if-needed → invoke → report) that auto-selects and sequences
  the most relevant available skill(s) for a request, plus the required
  "When & why to use" section and core rules (no self-recursion, specificity
  rule, don't force a skill, transparency).
- `references/matching-guide.md` — request-signal parsing, reading skill
  descriptions, a scoring table, the specificity rule, an overlap-disambiguation
  table, and when NOT to route.
- `references/orchestration.md` — gather→produce→review ordering, passing context
  between skills, loop avoidance, handling unavailable/failing/auth-gated skills,
  and reporting the routing.
- `LICENSE.txt` — MIT.

Source: user request — a skill that understands the prompt and automatically
selects/applies the relevant skills instead of choosing them manually.
